# Task 5 - Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis on the Titanic Dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
1. Loaded dataset
2. Checked dataset structure
3. Identified missing values
4. Generated statistical summary
5. Created histograms
6. Created boxplots
7. Performed bivariate analysis
8. Generated correlation heatmap
9. Created pairplot
10. Derived insights

## Key Findings
- Female passengers had higher survival rates.
- First-class passengers were more likely to survive.
- Fare positively influenced survival.
- Age distribution showed slight skewness.
- Fare contained several outliers.

## Outcome
Successfully identified patterns, trends, and relationships in the Titanic dataset through EDA.
